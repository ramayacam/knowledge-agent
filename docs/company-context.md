## Aspire Best Practices for CAM

### Contract Setup
✅ Use **Fixed Payment** for most regular service contracts  
✅ Separate opportunities by division/service type when billing varies  
✅ Use **T&M on Completion** for emergency response and special projects  
✅ Use **Fixed Price on Completion** for vacancy prep and construction cleanup  

### Service Tracking
✅ Create separate services for each major task (janitorial, floor care, landscape, etc.)  
✅ Use appropriate occurrences for frequency (daily = 260/year weekdays, weekly = 52/year, etc.)  
✅ Tag opportunities with industry type for reporting  

### Work Ticket Management
✅ Complete tickets promptly for accurate billing  
✅ Use Dynamic Forecasting for large contracts with seasonal variations  
✅ Document special projects and emergency work thoroughly  

### Invoicing
✅ Link completed tickets to invoices for transparency (especially for retail chains)  
✅ Provide detailed breakdowns for property managers  
✅ Use consistent invoice timing (1st of month standard)

## Example Customer Questions & CAM Aspire Solutions

### Question: "We're a new CVS location. How do I set up our contract?"
**CAM Solution in Aspire:**
1. Create Property for the CVS location
2. Create Contract Opportunity (Fixed Payment)
3. Add services:
   - Nightly janitorial (260 occurrences for weekdays)
   - Weekly floor care (52 occurrences)
   - Monthly power washing (12 occurrences)
4. Configure Fixed Payment schedule for 12 equal monthly payments
5. Send proposal with detailed service breakdown

### Question: "We need emergency sanitization after an incident."
**CAM Solution in Aspire:**
1. Create Work Order for the property
2. Select T&M on Completion invoice type
3. Add services:
   - Emergency sanitization
   - Level 2 cleaning with EPA-approved virucide
   - Documentation and reporting
4. Mark as Won, create work ticket
5. Schedule same-day response
6. Complete ticket with actual time and materials
7. Invoice based on actual costs

### Question: "Our shopping center needs vacancy prep for 3 stores."
**CAM Solution in Aspire:**
1. Create Work Order for shopping center property
2. Select Fixed Price on Completion
3. Create separate services for each store:
   - Store A: Demolition + cleanup
   - Store B: Demolition + cleanup
   - Store C: Demolition + cleanup
4. Or: Single service with separate work tickets
5. Price based on square footage and scope
6. Invoice when all work is complete
